3

Set Up Telegram Notifications

Stock Alerts Pro sends price alerts via Telegram. You need to get your personal Chat ID and add it to the app. This takes about 2 minutes.

📱 Part A — Get Your Telegram Chat ID

1

Open Telegram

Install Telegram if you don't have it — available free on the App Store and Google Play.

2

Search for @userinfo3bot

In the Telegram search bar, type @userinfo3bot and open the bot.

3

Send any message

Type anything and send it. The bot will immediately reply with your details including your Chat ID — a number like 7327563441

4

Copy the Chat ID number

Long press on the number and copy it. You'll need it in the next step.

⚙️ Part B — Add Chat ID to Stock Alerts Pro

1

Start the Stock Alerts Bot on Telegram

Open Telegram, search for @SPA2026bot and tap START. This step is required — the bot cannot send you messages until you start it.

2

Go to Settings

In Stock Alerts Pro, tap the ⚙️ Settings icon in the top menu.

3

Find "📱 Telegram Notifications"

Scroll down to the Telegram Notifications section.

4

Enable notifications and paste your Chat ID

Tick "Enable Telegram Notifications" and paste your Chat ID number in the field.

5

Save Settings and send a test

Tap "💾 Save Settings", then tap "🧪 Send Test Notification". You should receive a Telegram message within seconds!

⚠️ Important: You must search for @SPA2026bot on Telegram and tap START before the bot can send you any messages. If you skip this step, notifications will silently fail.

4

Add Your First Stock Alert

1

Tap "➕ Add Alert"

Tap the blue Add Alert button in the top navigation bar.

2

Enter the stock symbol

For US stocks: use the ticker symbol e.g. AAPL, TSLA, NVDA
For ASX stocks: add .AX suffix e.g. CBA.AX, BHP.AX

For NSE stocks: add .NS suffix e.g. RELIANCE.NS, TCS.NS

For BSE stocks: add .BO suffix e.g. RELIANCE.BO, INFY.BO

3

Set your target price and alert type

Enter the price you want to be alerted at. Choose Above (alert when price goes up to target) or Below (alert when price drops to target).

4

Save the alert

Tap Save Alert. Your alert appears on the dashboard and will monitor the price automatically!

5

Save to Your Home Screen

⚠️ Important: Make sure you are logged in and can see your Dashboard before saving the shortcut. This ensures the app opens directly to your dashboard next time.

🍎 iPhone / iPad (Safari only)

1

Open in Safari

Make sure you're using Safari — this won't work in Chrome on iPhone.

2

Tap the Share button

Tap the Share icon at the bottom of the screen — it looks like a box with an arrow pointing up ⬆

3

Tap "Add to Home Screen"

Scroll down in the share menu and tap "Add to Home Screen".

4

Confirm the name and tap "Add"

The name will show as Stock Alerts Pro. Tap Add in the top right corner.

💡 Result: The bell icon will appear on your iPhone home screen. Tapping it opens the app in full-screen mode — just like a native app with no browser address bar!

🤖 Android (Chrome)

1

Open in Chrome

Make sure you're using Google Chrome browser.

2

Tap the three-dot menu

Tap the ⋮ menu icon in the top right corner of Chrome.

3

Tap "Add to Home screen"

Tap "Add to Home screen" from the menu.

4

Tap "Add"

A prompt will appear — tap Add to confirm. The icon will appear on your home screen.

💡 Note: On some Android devices, Chrome may show an automatic "Add to Home screen" banner at the bottom of the screen — just tap that!
6

Save Shortcut to Desktop (Computer)

🖥️ Windows — Chrome or Edge

1

Open the app in Chrome or Edge

Visit the app URL while logged in to your dashboard.

2

Click the install icon in the address bar

Look for a ⊕ or computer icon on the right side of the address bar. Click it and select "Install".

3

Alternatively: use the browser menu

Click the ⋮ menu → "Save and share" → "Install page as app" (Chrome) or "Apps" → "Install this site as an app" (Edge).

🍎 Mac — Safari

1

Open in Safari and go to dashboard

Make sure you're logged in.

2

Click File → Add to Dock

In the menu bar click File → "Add to Dock" (macOS Sonoma and later).

3

Or drag the URL to your desktop

Click and drag the padlock icon in the address bar to your desktop to create a shortcut.
